# iOS_OTT

## NetFlix 홈 영화 리스트 Demo Project  

1. 개발 언어: Swift 
2. 적용 패턴: CleanSwift
3. 구현 설명: Movie Open API를 활용하여 영화 데이터 수집 및 가공, 
   각 영화 콘텐츠 목록은 다른 타입의 데이터를 수집하였고, 각각 리스트는 페이징 가능하게 구현되어있음
